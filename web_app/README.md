# Web Application

This folder contains the web application built to display the results of each step of the FICO Educational Challenge 2026.

## Tech Stack
- **Backend:** Flask (Python)
- **Database:** MySQL
- **Hosting:** [PythonAnywhere](https://www.pythonanywhere.com/)

## Features
- Displays results from each phase of the challenge.
- Connects to a MySQL database to fetch and present data/metrics.
